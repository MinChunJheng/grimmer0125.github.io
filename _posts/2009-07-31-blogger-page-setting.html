--- title: 修改google blog版面區塊大小 date: '2009-07-31T10:55:00.016+08:00' author: Teng-Chieh Kang tags: - blog modified_time: '2010-05-21T22:41:22.354+08:00' blogger_id: tag:blogger.com,1999:blog-2031747454610917099.post-1568696159462670148 blogger_orig_url: http://q-and-g.blogspot.com/2009/07/blog.html --- 這是我花了一點點時間研究出來的，記錄一下。
假設一開始在blogspot首頁，選自訂->版面配置->修改html。

1.修改blog標題寬度。

#header-wrapper {
   ...
   width:660->960px;
   ...
}

2.修改文章區塊+側邊欄總寬度。

#outer-wrapper {
   ...
   width: 660->960px;
   ...
}

3.修改文章區塊寬度。

#main-wrapper {
   ...
   width: 410->710px;
   ...
}

4.修改blog標題描述的文字寬度。

#header .description {
   ...
   max-width:700->1000px;
   ...
}
